Introduction

This section contains test implementations of th scoreFollowsAudio Javascript Library. At the moment the following test are implemented as examples as follows:

facsimile.html
implements a facsimile page turn demo using a recording and autogaph images derived from the Freischütz Digital project
verovio-test.html
test verovio with an example copied from the verovio website
rendering.html
impleents a rendered score (using verovio) of No. 6 from Weber's Opera Der Freischütz based on an encoding derived from the Freischütz Digital project
